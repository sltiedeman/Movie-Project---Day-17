Lets Find a Movie

This site uses the Open Movie Database API to allow a user to search for movies/actors/actresses. On page load the user sees all of the currently playing movies. Through a simple search they can look for a wide array of movies/actors/actresses that are included in the database. A click on the provided images opens a modal window for additional information

The Open Movie Database has some holes in its data where images are not provided for more obscure movies and actors/actresses. A default blank image is used to leave the site aesthetically pleasing.
